# Hybrid_DB_Project Frontend

This is the frontend directory for the Hybrid_DB_Project.

## Setup

Firebase is already installed and configured in `firebase-config.js`.

## Next Steps

1. Set up your frontend framework (React, Vue, etc.)
2. Import and use the Firebase config:
   ```javascript
   import { app, analytics } from './firebase-config.js';
   ```

3. Connect to the backend API at `http://localhost:4000`

## Firebase Configuration

The Firebase configuration is already set up in `firebase-config.js` with your project credentials.
